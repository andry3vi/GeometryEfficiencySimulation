# GeometryEfficiencySimulation

install with:

`pip install .`

launch with:

`ges -h`

optional arguments:
-  -h, --help\
    show this help message and exit
-  -ev EV     \
    Number of event to be generated
-  -graphic   \
    plot graphical results
-  -S SDIAMETER\
    Source diameter
-  -D DDIAMETER\
    Detector diameter
-  -DS DSDISTANCE\
  Detector Soruce distance
